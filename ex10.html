(검색) javascript object iterate....for(var key in object){}
함수정의 coworkers.showAll=function(){}..은 function showAll(){}과 같은 의미
showAll()함수 내 있는 coworkers가 다른곳에 있는 동일한 이름의 객체이와 충돌 가능성을 피하는 방법으로,
showAll() 함수 내에서 함수에 소속된 객체를 가르키는 약속된 기호 this를 사용
# javascript의 this가 해당함수 호출 패턴에 다라 객제참조하는 규칙
  1. 기본적으로 this는 전역객체
  2. 메소드 내부의 this는 해당 메소드를 호출한 부모객체를 참조
  3. 생성자 함수 코드 내부의 this는 새로 생성된 객체를 참조
  4. call()과 apply() 메소드로 함수를 호출할때, 함수의 this는 첫번째 인자로 넘겨받은 객체를 참조
  5. 프로토타입 객체 메소드 내부의 this도 해당 메소드를 호출한 부모 객체를 참조
  6. javascript의 this 키워드는 접근제어자 public 역할을 한다.

하지만 이경우 showAll도 coworkers에 소속된 대상이므로 함수까지표시되는 문제가 생김 --> if 문 사용하여 제거방법 --> if(key !="showAll"){} 추가 coworkers.showAll = function(){ for(var key in this){ if(key !="showAll") { document.write(key+':'+this[key]+'
'); } } }
(용어) 객체에 소속된 함수는 메소드(method), 객체에 소속된 변수는 프로퍼티(property)
